Summary#

Summative Proctor UI#

Built a React-Redux user interface for proctoring (running) student tests.

Powerful tests
The UI used algebraic data types with Flow instead of ordinary unit tests. Automated tests are Cucumber + Nightwatch.js to pilot a browser using tests that relatively untrained QA members could contribute to. Achieved 100% type coverage, and 350+ acceptance test scenarios.
Automatic Swagger synchronization
Mocks were kept in sync with the Swagger definitions, and all data was sanitized at virtually no cost. The UI oftentimes was the insurance that APIs stayed true to their contract.
